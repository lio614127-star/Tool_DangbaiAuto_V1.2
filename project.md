# Project: Douyin Downloader & Analyzer

## Overview
A desktop application built with Python (CustomTkinter) and Playwright designed to analyze Douyin channels and download videos by injecting custom JavaScript into the browser console.

## Mission
To provide a high-performance, premium-look tool for content creators to analyze viral trends on Douyin and conveniently manage video downloads without manual browser interaction.

## Core Tech Stack
- **Frontend**: CustomTkinter (Python)
- **Engine**: Playwright (Headless Chromium)
- **Logic**: Python 3.10+
- **Data Extraction**: Custom JavaScript Injection (Provided by User)
